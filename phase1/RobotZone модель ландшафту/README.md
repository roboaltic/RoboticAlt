# Інструкція для встановлення карти з текстурами (UE4)

Цей репозиторій містить карту та текстури для **Unreal Engine 4 (UE4)**. Нижче описано, як користувач може встановити та запустити ці файли.

## Вимоги

1. **Unreal Engine 4** (версія 4.27.2) встановлена на вашому комп'ютері.
2. Структура папок повинна залишатись незмінна для коректної міграції файлів в інший проект
   ## Додавання карти у Unreal Engine 4

Щоб карта з текстурами працювала у вашому проекті UE4, виконайте такі кроки:

1. **Відкрийте Unreal Engine 4**  
   Переконайтеся, що у вас встановлена версія UE4, сумісна з картою (рекомендовано 4.27.2).

2. **Створіть або відкрийте проект UE4**  
   - Для нових користувачів: створіть **Blank Project** (без стартового контенту).  
   - Для існуючого проекту переконайтеся, що версія UE4 збігається.

3. **Скопіюйте файли карти та текстур у папку Content**  
   - Структура повинна бути такою:
     ```
     REPO/
     ├─ Map/
     │  └─ ThirdPersonExampleMap.umap
     ├─ Textures/
     │  └─ *.uasset
     ```
   - Скопіюйте папки `Map` і `Textures` у папку `Content` вашого UE4-проекту:
     ```
     MyUE4Project/
     └─ Content/
         ├─ Map/
         └─ Textures/
     ```

4. **Оновіть Assets у UE4**  
   - В UE4 відкрийте **Content Browser** → натисніть **File → Refresh All Assets**, щоб підхопити нові `.umap` та `.uasset` файли.

5. **Відкрийте карту**  
   - У Content Browser знайдіть папку `Map` і двічі клікніть на файл `.umap`.  
   - Карта завантажиться з усіма текстурами.

6. **Примітки**
   - Не змінюйте назви файлів `.umap` і `.uasset`.  
   - Якщо текстури не з’являються, перевірте, що вони знаходяться у правильній папці `Textures`.
