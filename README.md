##  Конструкція стенду для відпрацювання розподіленого керування роботом

## Загальна характеристика

Даний репозиторій містить матеріали та програмні розробки, виконані в межах практичної підготовки з використанням сучасних апаратно-програмних засобів автоматизації, робототехніки та серверних систем. Проєкт спрямований на формування практичних навичок у сфері проєктування технічних стендів, мобільних роботизованих платформ та інформаційних інфраструктур.

Робота виконувалася поетапно та охоплює три основні фази: створення навчального стенду, розробку мобільного робота та налаштування комп’ютерних і серверних систем.

---

## ФАЗА 1. МЕХАНІЧНИЙ СТЕНД

### Мета фази

Перший етап спрямований на створення навчального технічного стенду для дослідження принципів автоматизації, електроніки та керування апаратними компонентами. Основною метою є формування базових практичних навичок роботи з електричними схемами, мікроконтролерами та периферійними пристроями.

### Основні завдання

У межах даної фази було виконано аналіз можливих варіантів побудови стенду, визначено перелік необхідних компонентів та спроєктовано загальну схему підключення. Особливу увагу приділено питанням електроживлення, захисту елементів та стабільності роботи системи.

Проведено монтаж апаратної частини, підключення сенсорів, виконавчих механізмів, блоків живлення та комунікаційних модулів. Паралельно здійснювалося налаштування програмного забезпечення для тестування окремих компонентів.

Було реалізовано базові сценарії роботи стенду, зокрема керування навантаженнями, зчитування даних із сенсорів та передавання інформації до комп’ютера.

## ФАЗА 2. ROS2 + РОБОТ

### Мета фази

Другий етап спрямований на створення мобільної роботизованої платформи та її інтеграцію з програмним середовищем керування. Основною метою є формування навичок роботи з робототехнічними системами, сенсорикою та розподіленими обчисленнями.

### Проєктування апаратної частини

На даному етапі було спроєктовано конструкцію мобільного робота з використанням двигунів постійного струму, драйверів керування, плати Raspberry Pi та периферійних пристроїв. Особливу увагу приділено механічній стабільності, балансуванню платформи та оптимальному розміщенню компонентів.

Здійснено підключення моторів, камери, джерел живлення та комунікаційних інтерфейсів. Проведено тестування роботи апаратної частини у різних режимах навантаження.

### Програмна реалізація

Для програмного керування роботом використано фреймворк ROS 2. Було розроблено окремі програмні вузли для керування рухом, обробки відеопотоку та взаємодії з користувачем.

Реалізовано диференціальну модель руху, підтримку команд типу `/cmd_vel`, публікацію одометрії та передачу зображень з камери. Забезпечено стабільну взаємодію між компонентами через механізми topics та QoS.

Окрему увагу приділено оптимізації роботи моторів, компенсації механічного опору та синхронізації запуску системи.


## ФАЗА 3. КОМП'ЮТЕРИ ТА МЕРЕЖА

### Мета фази

Третій етап присвячений формуванню інформаційної інфраструктури для підтримки роботи стенду та робота. Основною метою є набуття навичок адміністрування операційних систем, налаштування мережевих сервісів та серверного програмного забезпечення.

### Конфігурація робочих станцій

На даному етапі виконано встановлення та налаштування операційних систем Linux на персональних комп’ютерах. Проведено конфігурацію користувацьких середовищ, встановлення необхідних пакетів та інструментів розробки.

Здійснено налаштування SSH-доступу, систем оновлення та резервного копіювання. Забезпечено стабільну взаємодію між робочими станціями та роботизованою платформою.

### Мережева інтеграція

Виконано налаштування локальної мережі для обміну даними між комп’ютерами, сервером та роботом. Забезпечено коректну маршрутизацію, синхронізацію часу та стабільність з’єднання.

Окрему увагу приділено оптимізації пропускної здатності для передачі відеопотоку та телеметрії.
