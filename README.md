##  Конструкція стенду для відпрацювання розподіленого керування роботом

## Загальна характеристика

Даний репозиторій містить матеріали та програмні розробки, виконані в межах практичної підготовки з використанням сучасних апаратно-програмних засобів автоматизації, робототехніки та серверних систем. Проєкт спрямований на формування практичних навичок у сфері проєктування технічних стендів, мобільних роботизованих платформ та інформаційних інфраструктур.

Робота виконувалася поетапно та охоплює три основні фази: створення стенду, розробку робота та налаштування комп’ютерних і мережі.

## ФАЗА 2. ROS2 + РОБОТ

### Мета фази

Другий етап спрямований на створення мобільної роботизованої платформи та її інтеграцію з програмним середовищем керування. Основною метою є формування навичок роботи з робототехнічними системами, сенсорикою та розподіленими обчисленнями.

### Проєктування апаратної частини

На даному етапі було спроєктовано конструкцію мобільного робота з використанням двигунів постійного струму, драйверів керування, плати Raspberry Pi та периферійних пристроїв. Особливу увагу приділено механічній стабільності, балансуванню платформи та оптимальному розміщенню компонентів.

Здійснено підключення моторів, камери, джерел живлення та комунікаційних інтерфейсів. Проведено тестування роботи апаратної частини у різних режимах навантаження.

### Програмна реалізація

Для програмного керування роботом використано фреймворк ROS 2. Було розроблено окремі програмні вузли для керування рухом, обробки відеопотоку та взаємодії з користувачем.

Реалізовано диференціальну модель руху, підтримку команд типу `/cmd_vel`, публікацію одометрії та передачу зображень з камери. Забезпечено стабільну взаємодію між компонентами через механізми topics та QoS.

Окрему увагу приділено оптимізації роботи моторів, компенсації механічного опору та синхронізації запуску системи.
