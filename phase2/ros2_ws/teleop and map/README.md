
## keycontrol.py
нода `keyboard_teleop` призначений для ручного керування мобільним роботом за допомогою клавіатури з використанням термінального інтерфейсу. Даний модуль забезпечує формування керувальних команд у реальному часі на основі дій оператора та їх передачу в систему керування рухом.

## route_executor.py
- нода яка надає можлтвість задати маршрут в `route.yaml`. В такому випадку робот виконує задані дії автоматично
