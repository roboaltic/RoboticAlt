## motor_driver
Пакет `motor_node` призначений для реалізації керування рухом мобільного робота з диференціальним приводом на основі драйвера L298N та бібліотеки керування GPIO. Даний модуль забезпечує перетворення високорівневих команд руху, представлених у вигляді повідомлень ROS 2, у низькорівневі керувальні сигнали для електродвигунів постійного струму.
