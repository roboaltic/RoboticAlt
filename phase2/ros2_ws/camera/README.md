## camera_node.py
Пакет camera_driver призначений для інтеграції відеокамери в робототехнічну систему на основі ROS 2 та забезпечує отримання, обробку і передачу зображень у реальному часі. Даний модуль виконує функцію програмного інтерфейсу між фізичним пристроєм відеозахоплення та програмною інфраструктурою робота, що дозволяє використовувати візуальну інформацію для навігації, аналізу середовища та контролю роботи системи.
