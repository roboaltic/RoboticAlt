## sample_launch.py

Файл `sample_launch.py` призначений для демонстрації базового механізму запуску нод у середовищі ROS 2. Його основна функція полягає у відображенні принципів декларативного опису процесів, параметрів та взаємодії компонентів у межах launch-системи. Даний файл використовується як тестовий або навчальний приклад, що дозволяє перевірити працездатність окремих нод і коректність їх інтеграції без повного запуску всієї робототехнічної платформи.

Він виконує допоміжну роль і слугує інструментом для налагодження, експериментів та перевірки конфігурацій перед використанням основного сценарію запуску.

## robot_launch.py

Файл `robot_launch.py` є основним координаційним сценарієм запуску всієї системи мобільного робота. Його призначення полягає у централізованому старті взаємопов’язаних нод, зокрема модуля керування моторами, драйвера камери, системи ручного керування або автономного виконання маршруту.

У межах launch-файлу визначаються параметри запуску, передача конфігурацій до нод, а також їх синхронізований старт. Такий підхід забезпечує узгоджене функціонування компонентів системи та спрощує процес ініціалізації робототехнічного комплексу однією командою.

Таким чином, `robot_launch.py` реалізує інтеграційний рівень програмної архітектури та формує завершене робоче середовище для експлуатації мобільного робота.
